## Getting Started

To play the game, follow these steps:

1. **Prerequisites:** Ensure you have Python and Pygame installed on your system.

2. **Clone the Repository:** Clone this repository to your local machine using Git:

   ```
   git clone https://github.com/your-username/raining-cats-and-dogs.git
   ```

3. **Run the Game:** Navigate to the game's directory and run the following command:

   ```
   python game.py
   ```

4. **Enjoy:** Play the game and try to catch as many cats and dogs as you can while avoiding their poop!

## Gameplay

In "Raining Cats and Dogs," you control a character at the bottom of the screen. Cats and dogs fall from the sky, and your goal is to catch them while avoiding falling objects.

- **Cats:** Catching cats earns you points.
- **Dogs:** Catching dogs earns you points.
- **Poop:** Avoid these objects, as they slow you down.

The game ends when you lose too many pets and you run out of lives. Try to achieve the highest score and beat your previous high score!

## Controls

- Use the **left arrow key** to move your character left.
- Use the **right arrow key** to move your character right.
